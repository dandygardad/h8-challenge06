# Challenge 06 (Hacktiv8 FGA)

## Deskripsi Project
Project ini berfokus dalam pembuatan simple REST API untuk penyimpanan data-data buku menggunakan framework Gin. Dalam API ini memiliki 5 method yaitu: 
- Get All Books
- Get Book by ID
- Create Book
- Update Book
- Delete Book
